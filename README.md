# 0021-drum-machine 



This was a project in which I build a drum machine for mixing sounds and creating music.  
I used: 
- React
- SASS 
- JavaScript
- HTML 




### Development Notes: 

Run npm: 
```bash 
npm start
```


To run SASS on watch mode run: 

```bash
npx sass src/style/main.sass src/style/main.css -w
````
